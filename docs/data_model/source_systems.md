# Source Systems

## 1. Property Registration System

Stores registered rental properties, landlords, property type, county, Eircode, registration status, and registration dates.

## 2. Rental / Tenancy System

Stores tenancy records including rent amount, tenancy start date, property, landlord, tenant, rent frequency, and tenancy status.

## 3. Dispute / Case Management System

Stores complaints and disputes raised by tenants or landlords, including case type, status, priority, opened date, closed date, and assigned officer.

## 4. Reference Data System

Stores controlled reference values such as county, property type, rent frequency, case type, case status, and registration status.
